## 🔧 Setup

1. Clone the repository:
```bash
git clone https://github.com/Qethys/money-maker-bot.git
cd money-maker-bot
```

2. Install dependencies:
```bash
npm install
pip install PyGithub requests
```

3. Create `.github_token` file with your GitHub token

4. Run the bot:
```bash
# JavaScript version
node money-maker-bot.js

# Python version  
python3 money-maker-bot.py

# With PM2 (recommended)
npm install -g pm2
pm2 start money-maker-bot.py --interpreter python3
```
